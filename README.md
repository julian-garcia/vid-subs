# VidSubs
Facilitate addition of subtitles to YouTube videos
***

## Components
- Video viewport
- Subtitle Entry
- Time navigator
- Video Library
  
## Services
- Storage - generate/append/read vtt + video url
- Backend - add/update/remove vtt + video url
- Download vtt file
  
## Utilities
- Navigate back/forth
- Set start and end markers
- Navigate video to selected subtitle
